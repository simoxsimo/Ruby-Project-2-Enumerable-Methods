## Description of the project 

The purpose of this project is to implement the following Enumerable Methods:
*  each
*  each_with_index
*  select
*  all?
*  any?
*  none?
*  count
*  map
*  inject

## Additional points
```
Hashes are also included in "my_" custom methods
```
